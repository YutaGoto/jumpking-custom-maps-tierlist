# Jump King Custom Maps Tierlist

ジャンプキングというゲームのカスタムマップのティアリストを作成できるページを作ります。

## 技術要件

- パッケージ管理はpnpm
- フロントエンドはNext.jsでなくてもよい。1ページしか用意しない想定なので、軽いものがよい。
- CSSはTailwindCSSを導入する。
- デプロイ先はvercelを考えている。

## 機能要望

- 候補となるマップはjsonで管理する。（具体的な内容はあとで用意する）
- マップを自由にドラッグ＆ドロップで並び替えられるようにする。
- ティアは1-20まで用意する
- 作成者名を入力できる。
- できあがったティアリストを画像でダウンロードできるようにする。
- Cookieで状態を保存できるようにする。
