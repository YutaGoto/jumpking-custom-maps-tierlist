# Jump King Custom Maps Tierlist Creator

A professional, high-performance web application for creating and sharing tier lists of custom maps for Jump King. Built with React 19, Tailwind CSS v4, and dnd-kit.

## Features

- **Dynamic Tier Management**: Add, delete, reorder, and customize the color of each tier.
- **Drag & Drop**: Smooth, accessible drag and drop functionality for organizing map thumbnails.
- **Image Export**: Export your high-quality tier list as a PNG image with a single click.
- **State Persistence**: Your progress is automatically saved to browser cookies.
- **Creator Name**: Personalize your exported tier list with your name.
- **Massive Map Database**: Includes over 130 pre-configured custom maps with local asset support and name fallbacks.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS v4
- **Drag & Drop**: @dnd-kit
- **Image Generation**: html-to-image
- **State Management**: React Hooks + js-cookie

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (recommended) or npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/jumpking-custom-maps-tierlist.git
   cd jumpking-custom-maps-tierlist
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Build for production:
   ```bash
   pnpm build
   ```
